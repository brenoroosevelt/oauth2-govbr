# Gov.Br OAuth 2.0 Client Provider

Cliente OAuth2 para GovBr.

## Instalação

```bash
composer brenoroosevelt/oauth2-govbr 
```

## Exemplo de Uso

```php
<?php
echo "ola mundo"; 
```

## Ambientes

```bash
cd [app-name]
composer start
```

Or using Docker: 
```bash
cd [app-name]
docker-compose up -d
```
Depois disso, abra `http://localhost:8080` em seu browser.
